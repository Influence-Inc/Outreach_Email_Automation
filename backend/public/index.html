Guidelines

Universal instructions for the AI when it reads creator replies and writes negotiation emails. Added to every AI prompt. (Outreach & follow-up emails are configured in Instantly.ai, not here.)

When this is off, every creator reply is flagged for a human at the top of the campaign activity list. When on, the AI uses the Guidelines below + few-shot examples to draft replies automatically.

Each reply type below shows the current master prompt Claude follows (view-only) and a text box where you tell it what to change. Write plain instructions (e.g. "keep it to 3 short lines", "always mention we can be flexible on timelines", "never use exclamation marks"). Your notes get added to the master prompt for that reply type. Leave a box empty to use the default. Deal terms and offer numbers are never changed from here — those come from the offer configurator.
